# Tune Console Variable

Right now, there are only two console variables, which can be used for testing and experimenting.

- `sar.DebugDraw`: Draw debug shapes for traces and viewport frustum.
- `sar.RealtimeLocationUpdate`: Don't wait until the viewport stops moving, update immediately.
